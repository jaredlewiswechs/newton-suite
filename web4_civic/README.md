# Web4 Civic

This is the Web4 Civic project, a local civic internet for Houston/Katy/Brookshire. The project is structured as follows:

- `client/`: The client application for users to interact with the system.
- `relay/`: The relay server for caching, indexing, and syncing cards.
- `law_engine/`: The engine for processing and applying Law Packs.

## Getting Started

1. Navigate to the respective directories to explore the components.
2. Refer to the `schemas/` and `examples/` directories for card definitions and examples.
3. Use the `laws/` directory for Law Pack definitions.
