# Teacher Guide — realTinyTalk Pack

This short guide helps teachers run a 4-lesson mini-course using TinyTalk and the realTinyTalk editor.

Audience: middle/HS CS students, coding clubs, workshops
Duration: four 45–60 minute sessions

Lesson 1 — Intro & Syntax (45 min)
- Goals: understand `let`, `law` (functions), `show`
- Activities: run `lessons/01_syntax_intro.tt` (simple greet, arithmetic, variables)
- Homework: modify a function to accept two params

Lesson 2 — Control Flow & Loops (45 min)
- Goals: practice `if`, `for in range`, `while`
- Activities: `lessons/02_control_flow.tt` (FizzBuzz, counters)

Lesson 3 — Data & Algorithms (60 min)
- Goals: lists, loops, basic algs
- Activities: `lessons/03_algorithms.tt` (map/filter/reduce examples)

Lesson 4 — Creative Demo (60 min)
- Goals: show creative coding in TinyTalk
- Activities: run `lessons/mandelbrot.tt` and `lessons/neighborhood.tt` demo
- Discussion: trade-offs, performance budgets, scaling

Assessment
- Small 1-page assignment: implement a small simulation (5–10 lines)

Resources
