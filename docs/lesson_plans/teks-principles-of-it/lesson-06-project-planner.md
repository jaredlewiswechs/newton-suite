# Lesson 06: Project Planning & Collaboration

**Time:** 50 minutes  
**Focus:** Requirements, milestones, and teamwork.

## TEKS Alignment (Principles of Information Technology 130.272)
- **(c)(8)(A)** use project planning practices.
- **(c)(8)(B)** collaborate in a team environment.
- **(c)(8)(C)** evaluate project outcomes and revise.

## Objectives
Students will:
1. Break a project into milestones.
2. Assign tasks and track completion.
3. Update a plan based on feedback.

## Materials
- `docs/lesson_plans/teks-principles-of-it/06_project_planner.tt`

## Lesson Flow
### Warm-Up (5 minutes)
- “Why do software teams use checklists?”

### Mini-Lesson (10 minutes)
- Define requirements, milestones, and retrospectives.

### Guided Practice (15 minutes)
- Walk through `06_project_planner.tt` and find the task status logic.

### Independent Practice (15 minutes)
- Add a new milestone and print an updated summary.

### Exit Ticket (5 minutes)
- “What is one benefit of a sprint review?”
