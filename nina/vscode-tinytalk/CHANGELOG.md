# Changelog

All notable changes to the TinyTalk VS Code extension.

## [0.1.0] - 2026-02-04

### Added
- Initial release
- Syntax highlighting for TinyTalk (.tt, .tinytalk files)
- Code snippets for common patterns
- Run TinyTalk files (Ctrl+Shift+R)
- Run selected code
- Transpile to JavaScript
- Real-time syntax linting
- Language configuration (brackets, comments, folding)
